# Python-Web-Scraper-using-BeautifulSoup-Extract-Website-Data-with-Requests
This project demonstrates how to build a simple web scraper in Python to extract data from a website. The scraper sends an HTTP request to the webpage, downloads its HTML content, and parses the page using the BeautifulSoup library to extract specific information such as article titles.

The project scrapes headlines from Hacker News and prints them with numbering using Python’s enumerate() function.

This project is ideal for beginners who want to understand:

1. Web scraping fundamentals
2. HTML parsing using BeautifulSoup
3. Sending HTTP requests with Requests
4. Extracting structured data from websites
5. Automating data collection tasks in Python

## Features

1. Fetch webpage data using requests
2. Parse HTML using BeautifulSoup
3. Extract article titles automatically
4. Display results with indexing using enumerate()
5. Beginner-friendly and easy to modify

## Technologies Used

1. Python
2. Requests Library
3. BeautifulSoup (bs4)

## How to Run

1. Clone the repository
git clone https://github.com/your-username/web-scraper-project.git

2. Install dependencies
pip install requests beautifulsoup4

3. Run the program
